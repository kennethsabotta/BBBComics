# Placeholder Images

This folder should contain your comic page images in JPG format:

- cover.jpg (thumbnail for the library view)
- page-1.jpg
- page-2.jpg
- page-3.jpg
- etc.

Replace these placeholder files with your actual scanned comic pages.

## Image Guidelines

- **Format:** JPG
- **Recommended resolution:** 1200-2000px width for good quality on most screens
- **File naming:** Use sequential numbering (page-1.jpg, page-2.jpg, etc.)
- **File size:** Keep under 1-2 MB per image for fast loading
